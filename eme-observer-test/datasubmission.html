Data Submission

To add a new EME station to EME Observer or update an existing station entry, submit the station details to EME Directory maintained by Jan PA0PLY: https://www.pa0ply.nl/eme-directory/.

To include a station picture in the EME Observer station profiles, send the image to Max SA5IKN using the email address listed on his QRZ.com profile. Please specify what band the picture is for. Each band can use its dedicated picture or one picture can be used for all bands. An example of an EME Observer station profile with a picture is below.
